# TMDB Movie Analytics

Projet personnel d'extraction et d'analyse de données cinématographiques provenant de l'API TMDB.

## Technologies

- Python
- Pandas
- API REST
- Power BI

## Structure

- `src/extract.py` : récupération des données depuis TMDB
- `src/transform.py` : nettoyage et création des tables
- `data/raw` : données brutes
- `data/processed` : tables prêtes pour Power BI

## Statut

Projet en cours de développement.
